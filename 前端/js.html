this ​

this指向： this 永远指向最后调用它的那个对象。

1、左侧有对象，它就是 “this” 的引用。（隐式绑定）

2、用 “call”、“apply” 或者 “bind” 调用的，会显式地指明 “this” 的引用。（硬绑定）

3、用 “new” 调用，this” 指向的就是 JavaScript 解释器新创建的对象。（构造函数绑定）

4、严格模式”下，“this” 就是 undefined；否则“this” 会指向 “window” 对象。（默认绑定）

改变 this 的指向：

  • 使用 ES6 的箭头函数

    箭头函数的 this 始终指向函数定义时的 this，而非执行时。，箭头函数需要记着这句话：“箭头函数中没有 this 绑定，必须通过查找作用域链来决定其值，如果箭头函数被非箭头函数包含，则 this 绑定的是最近一层非箭头函数的 this，否则，this 为 undefined”。

  • 在函数内部使用 _this = this

  • 使用 apply、call、bind

  • new 实例化一个对象

浏览器里面，顶层对象是window，Node 里面，顶层对象是global，ES2020 在语言标准的层面，引入globalThis作为顶层对象 全局环境中，this会返回顶层对象。但是，Node.js 模块中this返回的是当前模块，ES6 模块中this返回的是undefined。 函数里面的this，如果函数不是作为对象的方法运行，而是单纯作为函数运行，this会指向顶层对象。但是，严格模式下，这时this会返回undefined。

apply,bind,call

1、都是用来改变函数的this对象的指向的。第一个参数都是this要指向的对象。都可以利用后续参数传参。

2、 call,bind传入的是若干个参数列表，apply接收的是一个包含多个参数的数组。

3、bind返回改定后的函数，方便我们后面调用，call、apply是对函数的直接调用

js
function mycall(ctx,...args){
    //保证fn的唯一性,避免误删原有属性
    let fn = Symbol(1);
    //将方法绑定到上下文
    ctx[fn]=this;
    //执行方法
    ctx.fn(args);
    //去除添加的fn
    delete ctx.fn;
}
function myapply(ctx,args=[]){
    if(args &&!(args instanceof Array())){
        throw('myapply只接受数组类型参数');
    }
    let fn = Symbol(1);
    ctx.fn=this;
    ctx.fn(...args);
    delete ctx.fn;
}
function mybind(ctx,...args1){
    return (...args2)=>{
        let fn = Symbol(1);
        ctx[fn]=this;
        ctx[fn](...args1.concat(args2));
        delete ctx[fn];
    }
}

function throttle(func,delay){
    let pre=0;

            pre = now;
        }

}

Reflect

Reflect 是一个内置的对象，它提供拦截 JavaScript 操作的方法。这些方法与proxy handlers的方法相同。Reflect不是一个函数对象，因此它是不可构造的。Reflect的所有属性和方法都是静态的。现阶段，某些方法同时在Object和Reflect对象上部署，未来的新方法将只部署在Reflect对象上。也就是说，从Reflect对象上可以拿到语言内部的方法。

class Promise {
  //构造方法
  constructor(executor) {
    //声明构造函数
    this.PromiseState = "pending";
    this.PromiseResult = null;
    this.callback = []; //保存回调函数
    let self = this;
    //resolve函数
    function resolve(data) {
      if (self.PromiseState != "pending") return;
      self.PromiseState = "fullfilled";
      self.PromiseResult = data;
      //执行成功的回调函数
      setTimeout(() => {
        self.callback.forEach((element) => {
          element.onResolved(data);
        });
      });
    }
    //reject函数
    function reject(data) {
      if (self.PromiseState != "pending") return;
      self.PromiseState = "rejected";
      self.PromiseResult = data;
      //执行失败的回调函数
      setTimeout(() => {
        self.callback.forEach((element) => {
          element.onRejected(data);
        });
      });
    }
    try {
      //同步执行执行器函数
      executor(resolve, reject);
    } catch (e) {
      reject(e);
    }
  }

  //then方法
  then (onResolved, onRejected) {
    //then方法返回值由指定的回调函数决定，回调返回非promise，then返回Promise，状态成功
    //回调返回promise，then返回毁掉的结果，状态未毁掉的状态
    let self = this;
    //判断回调函数参数
    if (typeof onRejected !== "function") {
      onRejected = (reason) => {
        throw reason;
      };
    }
    if (typeof onResolved !== "function") {
      onResolved = (value) => value;
    }
    return new Promise((resolve, reject) => {
      function callback(type) {
        try {
          let result = type(self.PromiseResult);
          if (result instanceof Promise) {
            result.then(
              (v) => {
                resolve(v);
              },
              (r) => {
                reject(r);
              }
            );
          } else {
            resolve(result);
          }
        } catch (e) {
          reject(e);
        }
      }
      if (this.PromiseState === "fullfilled") {
        setTimeout(() => {
          callback(onResolved);
        });
      }
      if (this.PromiseState === "rejected") {
        setTimeout(() => {
          callback(onRejected);
        });
      }
      if (this.PromiseState === "pending") {
        this.callback.push({
          onResolved: function () {
            callback(onResolved);
            // console.log('success');
          },
          onRejected: function () {
            callback(onRejected);
          },
        });
      }
    });
  };

  //catch方法
  catch (onRejected) {
    return this.then(undefined, onRejected);
  };

  //resolve方法
  static resolve(value) {
    return new Promise((resolve, reject) => {
      if (value instanceof Promise) {
        value.then(
          (v) => {
            resolve(v);
          },
          (r) => {
            reject(r);
          }
        );
      } else {
        reject(value);
      }
    });
  };
  //reject方法
  static reject(reason) {
    return new Promise((resolve, reject) => {
      reject(reason);
    });
  };
  //all方法
  static all(promises) {
    return new Promise((resolve, reject) => {
      let count = 0;
      let arr = [];
      for (let i = 0; i < promises.length; i++) {
        promises[i].then(
          (v) => {
            count++;
            if (count === promises.length) {
              resolve(arr);
            }
          },
          (r) => {
            reject(r);
          }
        );
      }
    });
  };
  //race方法

Async

定义异步函数(async function someName(){...})

  • 自动把函数转换为 Promise
  • 当调用异步函数时，函数返回值会被 resolve 处理
  • 异步函数内部可以使用 await

Await

暂停异步函数的执行 (var result = await someAsyncCall();)

  • 当使用在 Promise 前面时，await 等待 Promise 完成，并返回 Promise 的结果
  • await 只能和 Promise 一起使用，不能和 callback 一起使用
  • await 只能用在 async 函数中
