C/C++ ​

C 语言最初是用于系统开发工作，特别是组成操作系统的程序。由于 C 语言所产生的代码运行速度与汇编语言编写的代码运行速度几乎一样，所以采用 C 语言作为系统开发语言。

C11（也被称为C1X）指ISO标准ISO/IEC 9899:2011。在它之前的C语言标准为C99。

宏 ​

预处理指令就是去配置预处理器的各种各样的指令。一般预处理指令都会以“#”开头。宏的用途就是将一段代码替换成另一段，是处理代码文本上使用的。

c
#define MACRO_NAME macro_value
//MACRO_NAME 就是宏的名称。习惯上宏的命名和常量一样要全大写。
//macro_value 是宏的值。
#define WINDOW_WIDTH 1920 //代替全局常量
#define INT_LIST List<int> //代替 typedef
//宏函数
#define MAC_FUNCTION_NAME(PAR1, PAR2) value 
//当#加在一个参数的前面，它会被自动转换为那个给予的参数名的字符串
#define PRINTOUT_VALUE(var) std::cout<<"The value of "<<#var<<" is: "<<var<<std::endl;(void(0))
int a = 3;
PRINTOUT_VALUE(a);
The value of a is: 3
//两个双井号包裹住的中间的部分会从参数的形式转换成代码。
#define CALL_FUNC(func_num) func_##func_num##()
CALL_FUNC(1);
func_1()
//条件编译
#ifndef 和 #ifdef 是一种宏定义判断，作用是防止多重定义。
#ifndef 是 if not define 的简写
#ifdef 是 if define 的简写。
